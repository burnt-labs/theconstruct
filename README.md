# The Construct

> "I know kung fu." - Neo

The XION Engineering Team Playbook. Named after the training simulation in The Matrix, this is where we load up on how we work together.

## What's Inside

- **Roles** - What it means to lead a project
- **Processes** - How we update status, make decisions
- **Culture** - How we communicate, ask for help
- **Principles** - What we believe as engineers

## Development

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```
